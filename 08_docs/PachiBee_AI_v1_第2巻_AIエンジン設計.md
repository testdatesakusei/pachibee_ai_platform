# 1. AIエンジン全体構成

PachiBee AIは複数のAIで構成される。

```text

                    PachiBee AI

                          │

    ┌──────────┬──────────┬──────────┬──────────┐

    │          │          │          │

 MachineAI  MarketAI   HallAI   EventAI

    │          │          │          │

    └──────────┼──────────┼──────────┘

               │

       Prediction Engine

               │

         Recommendation AI

               │

          ChatGPT / Web / API

```

AIは独立して動作する。

それぞれがFeature Storeから特徴量を取得する。
