# PachiBee AI Platform 設計図

## 目的

- ベースExcelをデータベース化
- 機種ごとの稼働推移を分析
- パチンコ・パチスロ別にランキング作成
- 中古価格・設置台数・新台情報を追加
- OpenAI APIで根拠付きの予測文を作成
- 将来的にWebサイト・LINE Botへ展開

## 全体構成

1. データ収集
   - ベースExcel
   - 中古価格
   - 全国設置台数
   - 新台予定
   - 来店・取材

2. データベース
   - Supabase
   - 機種マスター
   - 週次稼働
   - 市場価格
   - AI予測
   - ホール分析

3. AI分析
   - 稼働予測
   - 中古価格予測
   - 売却タイミング
   - 機種評価
   - ホール評価

4. 表示
   - 管理画面
   - ユーザー向けWeb
   - ホール向けWeb
   - LINE Bot

## v1.0で作るもの

- ベースExcel取込
- パチンコ・パチスロ別ランキング
- CSV出力
- Supabase登録準備
