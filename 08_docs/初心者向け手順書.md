# 初心者向け手順書

## STEP 1：Cursorで開く

1. Cursorを開く
2. 左上の `File`
3. `Open Folder...`
4. この `pachibee_ai_platform` フォルダを選ぶ

## STEP 2：ベースExcelを入れる

1. パソコンでこのフォルダを開く
2. `base_files` フォルダを開く
3. 今までの `ベース2025...xlsx` や `ベース2026...xlsx` を全部入れる

## STEP 3：Supabaseにテーブルを作る

1. Supabaseを開く
2. `tokyo-slot-map` を開く
3. 左メニューの `SQL Editor`
4. `01_database/01_create_tables.sql` の中身を全部コピー
5. SupabaseのSQL Editorに貼る
6. `Run` を押す
7. 警告が出たら `Run and enable RLS`

## STEP 4：Excelを読む

Cursorのターミナルで以下を実行します。

```bash
pip install pandas openpyxl python-dotenv supabase
python 02_import/import_base_excel.py
```

## STEP 5：稼働予測を作る

```bash
python 04_ai_analysis/forecast_operation.py
```

結果は `output` フォルダに出ます。
